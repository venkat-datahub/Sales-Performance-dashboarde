# 📊 Sales Performance Dashboard

## 📌 Overview
This Power BI dashboard analyzes sales performance by region and year.
It helps identify trends and supports data-driven decision making.

## 📊 Key Insights
- APAC region has highest sales (40% contribution)
- Americas show steady performance
- Europe has lowest sales

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
